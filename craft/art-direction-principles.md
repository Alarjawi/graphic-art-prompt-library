# Art Direction Principles

## Principle 1: One Dominant Idea

Every image must answer one question:

```text
What should the viewer remember?
```

## Principle 2: Hierarchy Before Detail

Details are only valuable when the main idea is already clear.

## Principle 3: Context Creates Credibility

For Saudi work, context must be correct:

```text
architecture, clothing, light, landscape, civic spaces, signage behavior, cultural posture
```

## Principle 4: Production Language Beats Adjectives

Weak:

```text
make it premium
```

Stronger:

```text
controlled studio lighting, refined negative space, tactile materials, editorial composition, restrained color palette
```

## Principle 5: Negative Prompts Protect Taste

Negative prompts should remove likely visual failures, not randomly ban everything.
