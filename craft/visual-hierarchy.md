# Visual Hierarchy

## Purpose

Visual hierarchy controls what the viewer sees first, second, and third.

## Hierarchy Tools

- scale
- contrast
- focus
- spacing
- alignment
- lighting
- color accent
- depth
- isolation
- directional movement

## Prompt Cues

```text
dominant hero subject, secondary environmental context, tertiary texture detail, clear headline space, strong focal contrast, disciplined negative space
```

## Common Failures

- every object has equal importance
- background competes with subject
- no space for campaign message
- too many colors
- AI adds random text
- focal point is unclear

## Fix Prompt

```text
clear visual hierarchy, one dominant focal point, secondary background context, controlled supporting details, clean negative space for headline, no clutter, no competing focal points
```
