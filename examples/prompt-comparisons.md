# Prompt Comparisons

## Weak Versus Strong

### Weak

```text
make a Bauhaus poster
```

### Strong

```text
poster for a contemporary architecture lecture, Bauhaus-inspired graphic design, geometric composition using circles squares and triangles, primary red yellow blue with black accents, functional sans-serif typography, asymmetrical balanced layout, flat color planes, subtle screenprint texture, high clarity, no photorealism, no decorative flourishes, no glossy 3D
```

Why it is stronger:

- It names the design object.
- It describes composition.
- It gives a controlled palette.
- It includes typography direction.
- It blocks common AI mistakes.

## Prompt Upgrade Pattern

```text
[style name]
```

becomes:

```text
[design object] + [historical movement] + [composition] + [shape language] + [palette] + [typography] + [texture] + [negative prompt]
```
