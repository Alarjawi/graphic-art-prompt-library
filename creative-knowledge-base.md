# Creative Knowledge Base

## Mission

This repository is not a random prompt list. It is a structured art direction reference for building prompts that understand visual intent, market context, cultural accuracy, and production quality.

## Operating Model

Every strong prompt should pass through five layers:

1. Strategic intent
2. Visual world
3. Composition and hierarchy
4. Production language
5. Negative controls

## Strategic Intent

Clarify what the image must do.

```text
announce, inspire trust, communicate transformation, create prestige, launch a service, invite participation, dramatize innovation
```

## Visual World

Define the design universe.

```text
Saudi government premium, cinematic public-sector campaign, luxury identity, architectural realism, editorial key visual
```

## Composition And Hierarchy

Control how the eye moves.

```text
hero subject, foreground cue, midground activity, background context, clear focal point, disciplined negative space
```

## Production Language

Use real production terms.

```text
large-format campaign photography, cinematic lighting, controlled reflections, premium material finish, 85mm lens, soft atmospheric depth
```

## Negative Controls

Prevent AI drift.

```text
no generic stock photo, no fake Arabic text, no incorrect Saudi architecture, no plastic skin, no overprocessed HDR, no random logos
```

## Default Prompt Architecture

```text
[campaign object] for [audience/context], [strategic intent], [visual world], [subject/action], [Saudi/contextual accuracy], [composition], [lighting], [lens/camera], [materials/textures], [color system], [production finish], [negative controls]
```
