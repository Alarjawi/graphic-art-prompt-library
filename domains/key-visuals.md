# Key Visuals

## Direction

A key visual is the central image system for a campaign. It must be memorable, scalable, and able to carry adaptations across posters, social posts, outdoor, presentations, and motion.

## Core Requirements

- one dominant idea
- strong focal point
- adaptable negative space
- campaign world that can repeat
- clear foreground, midground, background
- visual metaphor when useful
- production polish

## Prompt Formula

```text
campaign key visual for [campaign], one clear central idea, strong hero focal point, scalable composition for outdoor and social formats, cinematic depth, clean negative space for headline, premium production finish, coherent color system, no clutter, no random text, no generic stock composition
```

## Quality Test

If the image is cropped into square, vertical, and wide formats, the idea should still survive.
