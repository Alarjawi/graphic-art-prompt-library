# Sources

This repository uses two kinds of sources.

## Trusted Art And Design Sources

Use these for historical claims and movement definitions.

- MoMA Art Terms: Bauhaus, Constructivism, De Stijl
  - https://www.moma.org/collection/terms/bauhaus
  - https://www.moma.org/collection/terms/constructivism
  - https://www.moma.org/collection/terms/de-stijl
- The Met Heilbrunn Timeline: Art Nouveau
  - https://www.metmuseum.org/essays/art-nouveau
- Tate Art Terms
  - https://www.tate.org.uk/art/art-terms
- Encyclopaedia Britannica: Graphic Design
  - https://www.britannica.com/art/graphic-design

## Practical Prompt And GitHub Sources

Use these for prompt structure ideas, keyword patterns, and AI image workflow language. Treat them as practical references, not historical authorities.

- MidJourney Styles and Keywords Reference
  - https://github.com/willwulfken/MidJourney-Styles-and-Keywords-Reference
- Fooocus SDXL styles
  - https://github.com/lllyasviel/Fooocus/tree/main/sdxl_styles
- AUTOMATIC1111 Stable Diffusion WebUI Wiki
  - https://github.com/AUTOMATIC1111/stable-diffusion-webui/wiki/Features
- Prompt Engineering Guide
  - https://github.com/dair-ai/Prompt-Engineering-Guide
- The Prompt Artists research paper
  - https://arxiv.org/abs/2303.12253

## Source Rule

If a source is from GitHub or a prompt community, use it for phrasing and workflow only. If a source is from a museum, university, encyclopedia, or design archive, use it for historical claims.
