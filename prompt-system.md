# Prompt System

## Core Principle

Do not rely on style labels alone. Use the label as context, then describe the visible design logic.

## Prompt Layers

### 1. Subject

What the image is about.

```text
a poster for an independent cinema festival
```

### 2. Format

What kind of graphic object should be made.

```text
editorial poster, album cover, packaging label, magazine spread, logo mark
```

### 3. Movement Or Style Language

The historical or design reference.

```text
Swiss Style, Bauhaus, Constructivism, Art Nouveau, De Stijl
```

### 4. Composition

How the page is arranged.

```text
modular grid, diagonal tension, centered emblem, asymmetric balance, orthogonal layout
```

### 5. Visual Vocabulary

The visible traits.

```text
flat geometric shapes, botanical linework, photomontage, hard black rules, clean whitespace
```

### 6. Color

Specific palette beats generic mood words.

```text
primary red yellow blue, black and cream, muted industrial red, metallic gold accents
```

### 7. Type And Print Cues

Useful for graphic work.

```text
condensed sans-serif typography, letterpress texture, halftone dots, screenprint ink, lithographic poster
```

### 8. Negative Prompt

Remove visual drift.

```text
no photorealism, no 3D render, no glossy mockup, no messy typography, no extra text
```

## Reusable Master Template

```text
[format] for [subject/audience], [movement]-inspired design, [composition], [visual vocabulary], [palette], [typography/layout cue], [print/material texture], high clarity, intentional negative space, no [unwanted traits]
```

## Verification Checklist

- The movement name is backed by a reliable source.
- The visual traits are observable, not just poetic.
- The prompt describes composition, color, type, and texture.
- The negative prompt removes likely mistakes.
- The final wording is specific enough to reproduce the style without copying one living artist.
