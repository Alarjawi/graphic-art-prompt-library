# Bauhaus

## Visual Logic

Bauhaus graphic language favors function, clarity, geometry, and modern industrial order. Prompts should emphasize simple shapes, rational layout, primary colors, and minimal ornament.

## Keywords

- geometric composition
- functional design
- primary colors
- circle square triangle
- sans-serif typography
- asymmetrical balance
- minimal ornament
- modernist poster
- flat color planes

## Prompt Formula

```text
[format] for [subject], Bauhaus-inspired graphic design, geometric composition using circles squares and triangles, primary red yellow blue with black accents, functional sans-serif typography, asymmetrical balanced layout, flat color planes, subtle screenprint texture, no photorealism, no ornament, no glossy 3D
```

## Example

```text
poster for a contemporary architecture lecture, Bauhaus-inspired graphic design, geometric composition using circles squares and triangles, primary red yellow blue with black accents, functional sans-serif typography, asymmetrical balanced layout, flat color planes, subtle screenprint texture, high clarity, no photorealism, no decorative flourishes, no glossy 3D
```
