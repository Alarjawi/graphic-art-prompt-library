# Swiss Style

## Visual Logic

Swiss Style, also called International Typographic Style, is useful for clean editorial systems, posters, identity systems, and information-led layouts. Prompts should emphasize grids, clarity, sans-serif typography, whitespace, and objective composition.

## Keywords

- modular grid
- asymmetric layout
- clean sans-serif typography
- objective design
- strong whitespace
- editorial hierarchy
- flush-left ragged-right text
- high contrast photography
- restrained palette

## Prompt Formula

```text
[format] for [subject], Swiss Style graphic design, strict modular grid, asymmetric editorial layout, clean sans-serif typography, generous whitespace, precise hierarchy, restrained palette with one accent color, crisp alignment, no ornament, no messy text, no decorative illustration
```

## Example

```text
museum exhibition poster for kinetic sculpture, Swiss Style graphic design, strict modular grid, asymmetric editorial layout, clean sans-serif typography, generous whitespace, black white and signal red palette, precise hierarchy, high contrast cropped photography, no ornament, no messy text, no decorative illustration
```
