# Constructivism

## Visual Logic

Constructivist-inspired graphic design often uses diagonal energy, industrial geometry, bold type, photomontage, and urgent red-black contrast. Prompts should feel dynamic and structural.

## Keywords

- diagonal tension
- photomontage
- bold red and black
- industrial geometry
- angular composition
- propaganda poster layout
- heavy sans-serif type
- strong contrast
- mechanical rhythm

## Prompt Formula

```text
[format] for [subject], Constructivist-inspired graphic design, diagonal tension, angular industrial geometry, bold red black and cream palette, photomontage elements, heavy sans-serif typography, dramatic scale shifts, screenprint texture, no soft gradients, no glossy render, no decorative elegance
```

## Example

```text
poster for an experimental theatre performance, Constructivist-inspired graphic design, diagonal tension, angular industrial geometry, bold red black and cream palette, photomontage fragments, heavy sans-serif typography, dramatic scale shifts, rough screenprint texture, no soft gradients, no glossy render, no decorative elegance
```
